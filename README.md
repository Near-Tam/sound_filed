# 设计的目的及意义
1.有助于定位缺陷位置、定量检测精度

2.将抽象的理论公式利用可视化技术进行分析

3.有助于提高检测精度和可靠性以及检测效率

4.形象直观表达了声场的特性

# 设计的具体内容
1.超声波的声场特性理论研究

2.圆形活塞换能器的辐射声场分布仿真模拟

3.矩形活塞换能器的辐射声场分布仿真模拟

4.圆形活塞换能器脉冲声场分布

5.GUI人机交互集成仿真的程序
